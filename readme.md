# Creating a Desktop App Using QT6

The tutorial:
    https://www.pythonguis.com/tutorials/pyqt6-creating-your-first-window/

## Installation:

install PyQt6 using:
> ```bash
> $ python3 -m pip install PyQt6
> ```
download: 8mb, 74mb


## run the app:
> ```bash
> $ python3 app.py
> ```

## run tests:

pytest tests/
# or
pytest tests/test_du_litera_silab_ilo.py -v
